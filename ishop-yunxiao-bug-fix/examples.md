## 工作项状态说明

| 状态名称 | 状态ID | 说明 |
|----------|--------|------|
| 待确认 | 28 | 新建Bug的初始状态 |
| 待处理 | 100005 | 已确认待开发处理 |
| 处理中 | 100010 | 开发人员正在修复 |
| 重新打开 | 30 | 验证未通过被重新打开 |
| 已修复 | 29 | 修复完成待验证 |
| 已关闭 | 100085 | 验证通过或无需修复 |

## 注意事项

1. **确认优先**：所有外部操作前会先向用户确认
2. **评论格式**：云效不支持 Markdown，评论使用纯文本
3. **责任范围**：仅处理分配给当前用户的 Bug
4. **状态流转**：某些状态更新可能受权限限制，需在 Web 端操作
